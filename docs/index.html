Msg

A javascript library that quickly enables modal windows with any html inside.

The window is centered and it grows or shrinks automatically by the content inside of it.

To close the window just click on the overlay that is created around it, or press Escape.


To install just include the file:
<script src="msg.js"></script>
Or the minified, smaller file:
<script src="msg-min.js"></script>
To use, first make an instance:
var msg = Msg();
To show a window just pass a string to the instance's show method:
msg.show("some string");
You can also pass an html element:
msg.show(document.getElementById("some-element"));

Structure

Four elements are created with the following ids:

"Msg-container-1" (with classes: "Msg-container" and "Msg-container-default")
"Msg-overlay-1" (with classes: "Msg-overlay" and "Msg-overlay-default")
"Msg-window-1" (with classes: "Msg-window" and "Msg-window-default")
"Msg-content-1" (with classes: "Msg-content" and "Msg-content-default")

The structure looks likes this: 
[container]
    [overlay][/overlay]
    [window]
        [content][/content]
    [/window]
[/container]
The instance id in this case is "1".

By default, ids are based on the number of instances created, it is incremental.

You can set custom ids and classes via options.

You can play with the ids and classes to have different types of windows.

To override the css you will probably need to use !important declarations.

The .show() or .set() functions can either receive a string, which would render any html inside of it, or an html element directly.

They change the html content of Msg-content elements.

msg.show("<i>Hello</i> <b>World</b>")

msg.show(document.getElementsByTagName("h1")[0])


The z-index values of the windows are generated when they are shown, allowing for windows with different ids to stack on top of each other.

Options

The following options can be passed inside an object when creating an instance:

id (Default: Number of instances): The id of the instance. For example, id:"foo".

class (Default: "default"): The class of the instance. For example, class:"bar".

container_class (Default: undefined): If this is defined, the Msg-container will use this instead of the class parameter.

overlay_class (Default: undefined): If this is defined, the Msg-overlay will use this instead of the class parameter.

window_class (Default: undefined): If this is defined, the Msg-window will use this instead of the class parameter.

content_class (Default: undefined): If this is defined, the Msg-content will use this instead of the class parameter.

lock (Default: true): Locks the background, prevents scrolling from propagating to the body. This is done through css by adding a class to the body temporarily.

close_on_overlay_click (Default: true): Closes the window when the overlay is clicked.

close_on_escape (Default: true): Closes the window when Escape is pressed.

clear_editables (Default: false): When pressing Escape, it clears the text from an active input=text or textarea element, if it has a value. If it's empty and close_on_escape is true, it closes the window.

before_show (Default: Empty Function): Function that triggers before the show method. Receives the instance as an argument. If false is returned it aborts the execution.

after_show (Default: Empty Function): Function that triggers after the show method. Receives the instance as an argument.

before_set (Default: Empty Function): Function that triggers before the set method. Receives the instance as an argument. If false is returned it aborts the execution.

after_set (Default: Empty Function): Function that triggers after the set method. Receives the instance as an argument.

before_close (Default: Empty Function): Function that triggers after the close method Receives the instance as an argument. If false is returned it aborts the execution.

after_close (Default: Empty Function): Function that triggers after the close method. Receives the instance as an argument.

before_create (Default: Empty Function): Function that triggers after the create method Receives the instance as an argument. If false is returned it aborts the execution.

after_create (Default: Empty Function): Function that triggers after the create method. Receives the instance as an argument.

before_destroy (Default: Empty Function): Function that triggers after the destroy method Receives the instance as an argument. If false is returned it aborts the execution.

after_destroy (Default: Empty Function): Function that triggers after the destroy method. Receives the instance as an argument.


Example: 
var msg_test = Msg(
{
    id: "test",
    class: "dark",
    overlay_class: "blur",
    after_show: function(instance)
    {
        instance.set(instance.html() + ", Mr. Roboto");
    },
    after_set: function(instance)
    {
        console.log("instance.content.innerHTML set to: " + instance.html());
    },
    before_close: function(instance)
    {
        return confirm("Are you sure?");
    }
});

msg_test.show("Domo Arigato");

Properties

instance.container: The Msg-container html element.

instance.overlay: The Msg-overlay html element.

instance.window: The Msg-window html element.

instance.content: The Msg-content html element.

instance.options: The options object of the instance.

Methods

instance.show(html): Opens a window with the received string or html element as its content and puts it at the top of the stack.

instance.show(): If no arguments are passed to show(), it displays the window with the last used html or the one passed to set() and puts it at the top of the stack.

instance.set(html): Updates the html of the window without opening it.

instance.close(): Closes the window.

instance.html(): A shortcut for instance.content.innerHTML. Returns "" if the html elements have not been created yet.

instance.is_open(): Returns true if the window is open.

instance.any_open(): Returns true if any window with any id is open.

instance.num_open(): Returns the number of windows open.

instance.num_instances(): Returns the number of instances created.

instance.create(): Creates the html elements if they don't exist yet. This is done automatically on the first show() or set(), but there might be cases where you might want to create them first without showing anything.

instance.recreate(): Removes the html elements from the dom and creates them again. This should be called after ids or classes are changed.

instance.destroy(): Removes the html elements from the dom.

instance.highest_zIndex(): Returns the highest z-index among all the created Msg-container elements. If it retuns -2000 it means no elements have been created.

instance.is_highest(): Returns true if the window is visible and at the top of the stack.

instance.to_top(): Puts an already opened window at the top of the stack. This is done automatically by show(), but there might be cases where you want to do this without triggering the show events.

instance.check_options(): This is done automatically when creating an instance. It checks the options object and applies the corresponding defaults to undefined properties.


Note: It's advised not to modify the "display" or "z-index" css properties of the Msg html elements since they are handled internally.
