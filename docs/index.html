Msg

A javascript library that quickly enables modal windows with any html inside.

The window is centered and it grows or shrinks automatically by the content inside of it.

To close the window just click on the overlay that is created around it.


To install just include the file:
<script src='msg.min.js'></script>
 To use, first make an instance:
var msg = Msg();
 To show a window just pass a string to the instance's show method:
msg.show('some string');

Three elements are created: "Msg-overlay-1" (with class "Msg-overlay"), "Msg-container-1" (with class "Msg-container") and "Msg-content-1" (with class "Msg-content").

The number at the end of the ids is based on the number of instances created, it is incremental. You can set a custom id via options.

To override the css you will probably need to use !important declarations.

Options

The following options can be passed inside an object when creating an instance:

id (Default: Number of instances): The id of the instance. For example, id:"foo" would create elements with the following ids: "Msg-overlay-foo", "Msg-container-foo" and "Msg-content-foo".

disable_scroll_propagation (Default: true): If this is kept enabled it will stop scroll propagation to parent elements inside and outside the window after reaching the top or bottom of an overflow.

Example: 
msg = Msg(
{
    id: "foo",
    disable_scroll_propagation: true
});

Properties

instance.overlay: The Msg-overlay html element.

instance.container: The Msg-container html element.

instance.content: The Msg-content html element.

Methods

instance.show(htmlstring): Shows a window with the received html string as its content.

instance.set(htmlstring): Updates the html of the container without opening the window.

instance.show(): If no arguments are passed to show(), it displays the window with the last used html or the one passed to set().

instance.close(): Closes the window.

instance.is_open(): Returns true if the window is open.

instance.any_open(): Returns true if any window with any id is open.

instance.num_open(): Returns the number of windows open.

instance.num_instances(): Returns the number of instances created.

instance.create(): Creates the html elements if they don't exist yet. This is done automatically on the first show() or set(), but there might be cases where you might want to create them first without showing anything.

instance.created(): Returns true if the html elements have been created.
